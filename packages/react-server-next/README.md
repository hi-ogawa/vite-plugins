# @hiogawa/react-server-next

Alias package for [`Next.js`](https://github.com/vercel/next.js)

## usage

```json
{
  "type": "module",
  "dependencies": {
    "@hiogawa/react-server": "latest",
    "next": "npm:@hiogawa/react-server-next@latest",
    "react": "rc",
    "react-dom": "rc",
    "react-server-dom-webpack": "rc"
  }
}
```
