## Quick start

See also [`./examples/starter`](./examples/starter).

```sh
# download template
npx degit@latest hi-ogawa/vite-plugins/packages/react-server/examples/starter my-project
cd my-project

# development
npm i
npm run dev

# build and preview
npm run build
npm run preview
```
